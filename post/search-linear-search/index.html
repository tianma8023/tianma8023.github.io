查找-顺序查找

发表于 2016-04-11 更新于 2019-07-31 分类于 Algorithm & Data Structure

定义

顺序查找又称为线性查找，其算法思路是从数组中的第一个（或最后一个）记录开始，将数组中元素逐个与需要查找的关键字进行比对，若发现有相等的，则查找成功；若始终未能相等，则查找失败。
